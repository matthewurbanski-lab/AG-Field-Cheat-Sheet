How the decision logic works (simple + editable)
The analyzer uses keyword rules. It does not replace inspection judgment. Add production-approved wording, thresholds, and exclusions as you refine.
  • Water / seepage → drainage + sump + discharge management
  • Humidity / musty → environment control (dehumidification) + sealing
  • Bowing wall / horizontal cracks → wall stabilization (bracing/anchors)
  • Sloped floors / bouncy → floor framing stabilization (posts/beams/joists)
  • Exterior brick cracks / door/window gaps → perimeter foundation stabilization (piers)
  • Uneven concrete → poly lift / slab support + water source control
